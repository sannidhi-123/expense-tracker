# 💰 Expense Tracker Web App

A simple and user-friendly web application built using HTML, CSS, and JavaScript to track daily income and expenses. It helps users manage their finances efficiently and monitor spending habits.

---

## 📌 Features

### 👤 User Functionalities

- ➕ Add income and expense transactions  
- 📊 View total balance, income, and expenses  
- 🧾 Transaction history display  
- 🗑️ Delete transactions  
- 💾 Data stored using browser local storage  
- ⚡ Real-time updates on UI  

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript  
- Storage: Browser Local Storage

## 🧪 Getting Started

### ✅ Prerequisites

- Any modern web browser (Chrome, Edge, Firefox)

---

### 🚀 Setup Instructions

1. Clone the repository

bash git clone https://github.com/sannidhi-123/expense_tracker.git 

2. Open the folder

bash cd expense_tracker 

3. Run the project

👉 Open index.html in your browser
